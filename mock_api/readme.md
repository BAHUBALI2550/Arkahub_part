# EnergyGrid Data Aggregator Assignment

This repository contains both:
- A mock backend server simulating the EnergyGrid API
- A Node.js client that aggregates telemetry from 500 devices

---

## Project Structure

- `mock-api/` – Mock EnergyGrid backend
- `client/` – Data aggregation client

---

## How to Run (Local)

### Terminal 1 – Start Mock API
```bash
cd mock-api
npm install
npm start
